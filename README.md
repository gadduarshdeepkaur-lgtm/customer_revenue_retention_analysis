# Customer Revenue & Retention Intelligence System

## Project Overview
This project analyzes customer transactions to identify revenue trends, customer retention patterns, and key business insights.

The analysis combines **Python, SQL, and Power BI** to create a complete data analytics workflow.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL (MySQL)
- Power BI
- Jupyter Notebook

## Dataset
Synthetic e-commerce dataset containing:
- Customers
- Products
- Orders
- Revenue metrics

## Key Analysis
- Revenue trend analysis
- Customer Lifetime Value (CLV)
- Customer churn analysis
- Cohort-based retention analysis
- Product performance analysis
- Geographic revenue analysis

## Dashboard
The Power BI dashboard provides interactive insights including:
- Revenue trends over time
- Revenue by product
- Revenue by city
- Top customers by revenue
- Key KPIs (Revenue, CLV, Churn Rate)

## Business Insights
- Laptop category generated highest revenue
- Pune generated highest regional sales
- Customer churn rate ≈ 41%
- High-value customers contribute majority of revenue
